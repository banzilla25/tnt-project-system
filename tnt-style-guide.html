14 — PATTERNS

Empty State

Saat data kosong, jangan biarkan layar polos. Beri ikon, satu kalimat arahan, dan tombol aksi agar tim tahu langkah berikutnya. Ini menggantikan teks abu tunggal yang ada sekarang.

Belum ada kreator

Mulai bangun roster dengan menambah kreator atau impor dari spreadsheet.

Belum ada riwayat aktivitas

Aktivitas tim akan tercatat otomatis di sini saat ada perubahan data.

16 — STANDARDS

Tier Kreator & Aturan Currency

Dua standar penulisan agar konsisten di seluruh app. Tier untuk klasifikasi roster, dan format Rupiah yang seragam antara tabel input dan kartu ringkasan.

Tier Badge
TIER A TIER B TIER C NANO

Tier A = prioritas tertinggi (gradient biru solid), B = menengah (biru lembut), C = standar (abu), NANO = micro-creator (ungu).
